MySQL

MySQL Database is a fully-managed database service, powered by the integrated HeatWave in-memory query accelerator.
It is the only cloud-native database service that combines transactions, analytics, and machine learning services into MySQL Database,
delivering real-time, secure analytics without the complexity, latency, and cost of ETL duplication.
It is developed, managed, and supported by the MySQL team in Oracle.

MySQL Database Service is available on Oracle Cloud Infrastructure, Amazon Web Service, and Oracle Database Service in Azure (ODSA).
The current documentation focuses on MySQL Database Service on Oracle Cloud Infrastructure.
For more information on MySQL Database Service on Amazon Web Services, see MySQL HeatWave, and on Oracle Database Service in Azure (ODSA), see Provisioning MySQL Heatwave.
